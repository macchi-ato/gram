## About

Gram is a small text editor for the terminal, inspired by the [Build Your Own Text Editor](https://viewsourcecode.org/snaptoken/kilo/) booklet, which itself is based on Antirez's [kilo](https://github.com/antirez/kilo) — a minimal text editor in roughly 1,000 lines of C.

This project is a learning exercise in systems programming, terminal I/O, and building software from scratch in C.
